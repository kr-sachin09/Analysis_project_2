# Mobile Sales Analysis Dashboard
This Power BI project analyzes the sales performance of a mobile retailer to track revenue against monthly targets. By connecting **Sales Transaction** data with **Target** sheets, the dashboard visualizes key insights like **Top Selling Models**, **Payment Preferences**, and **City-wise Sales**.

*Key Highlights:

* **Tech Stack:** Power BI, Data Modeling, DAX, & ETL.
* **Insights:** Tracks "Actual vs. Target" performance and customer buying behavior.
* **Visuals:** Interactive maps, bar charts, and funnels for easy decision-making.
